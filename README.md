# API
## Modules
- Node v5.0.0
- Express
- Express-Generator
- MySQL v5.7.13 (brew install mysql)

## Install
#### install dependencies:
` $ cd API && npm install `  

#### run the app:
` $ DEBUG=API:* npm start`
